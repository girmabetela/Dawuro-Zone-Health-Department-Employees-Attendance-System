# Dawuro Zone Health Department Employees Attendance System

## Overview
A digital attendance management system designed for Dawuro Zone Health Department employees.

## Features
- Employee registration and management
- Daily attendance tracking
- Check-in and check-out records
- Attendance reports
- Admin dashboard
- User authentication

## Technologies
- Frontend: React.js
- Backend: Node.js / Express
- Database: MySQL
